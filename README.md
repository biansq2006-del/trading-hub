# 交易工具台

这是一个 GitHub Pages 静态入口站，用来统一访问当前三个交易工具：

- 交易数据看板：https://biansq2006-del.github.io/stock_daily/
- 左侧交易策略：https://biansq2006-del.github.io/left-side-trading/left_index.html
- 主升浪交易策略：https://biansq2006-del.github.io/paper-trading/

## 部署到 GitHub Pages

推荐新建仓库 `trading-hub`，把本目录文件推送到仓库根目录，然后在 GitHub 仓库设置里开启 Pages：

1. Settings -> Pages
2. Build and deployment -> Source 选择 `Deploy from a branch`
3. Branch 选择 `main`，目录选择 `/root`

部署完成后页面地址通常是：

```text
https://biansq2006-del.github.io/trading-hub/
```

后续如果购买自定义域名，可以把这个入口站绑定为主域名首页。
